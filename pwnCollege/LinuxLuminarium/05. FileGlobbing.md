## Mixing globs
This was a fun challenge. I had to write the expression that would only showcase the three files.
```bash
hacker@globbing~mixing-globs:~$ cd /challenge/files/
hacker@globbing~mixing-globs:/challenge/files$ ls -a
.   amazing    challenging  educational  great  incredible  kind      magical  optimistic  queenly  splendid   uplifting   wonderful  youthful
..  beautiful  delightful   fantastic    happy  jovial      laughing  nice     pwning      radiant  thrilling  victorious  xenial     zesty
```
Now this got my thinking, on how I could formulate the perfect answer to get only these files and thus get the flag for this question.
```bash
hacker@globbing~mixing-globs:/challenge/files$ /challenge/run ^[cep]
Your expansion did not expand to the requested files (challenging, educational, 
pwning). Instead, it expanded to:
^[cep]
hacker@globbing~mixing-globs:/challenge/files$ /challenge/run [cep]*
You got it! Here is your flag!
pwn.college{UJHXkM2EsDiVkAp3e_YOQAxpiy0.dVjM4QDLyYjMxkzW}
hacker@globbing~mixing-globs:/challenge/files$ 
```
### Answer:
```
pwn.college{UJHXkM2EsDiVkAp3e_YOQAxpiy0.dVjM4QDLyYjMxkzW}
```
