## Searching Manuals
In the manual itself, we have to scroll and search  for the answer as we move up and down using the arrow keys, and searching for the flag using the searching `/` with the search pattern.
```bash
hacker@man~searching-manuals:~$ man challenge
```

```bash
hacker@man~searching-manuals:~$ /challenge/challenge --fvyn
```

```bash
hacker@man~searching-for-manuals:~$ man man
```
