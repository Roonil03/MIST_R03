# Shell Variables
## Printing Variables
This is a simple challenge of just `echo`-ing a variable...
```bash
hacker@variables~printing-variables:~$ echo $FLAG
pwn.college{886RFgPvW9c3Ba9DRDvw_akQdY8.ddTN1QDLyYjMxkzW}
hacker@variables~printing-variables:~$ 
```

```

```

## Exporting Variables
This challenge makes us tweak with variables with export
```bash
hacker@variables~exporting-variables:~$ /challenge/run 
Incorrect...
You have not exported the PWN variable. This challenge requires you to export 
the PWN variable.
hacker@variables~exporting-variables:~$ export PWN; PWN=COLLEGE; COLLEGE=PWN
You've set the PWN variable to the proper value!
You've set the COLLEGE variable to the proper value!
hacker@variables~exporting-variables:~$ /challenge/run 
CORRECT!
You have exported PWN=COLLEGE and set, but not exported, COLLEGE=PWN. Great 
job! Here is your flag:
pwn.college{0X7K-X-8cFdDpxn08OkjFc1kXit.dJjN1QDLyYjMxkzW}
You've set the PWN variable to the proper value!
You've set the COLLEGE variable to the proper value!
hacker@variables~exporting-variables:~$ 
```
### Answer:
```
pwn.college{0X7K-X-8cFdDpxn08OkjFc1kXit.dJjN1QDLyYjMxkzW}
```
